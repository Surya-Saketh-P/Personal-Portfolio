# Surya Saketh — Personal Portfolio

A minimalist, high-end personal portfolio built to showcase software engineering projects, technical skills, and interactive design. The site focuses on organic 3D motion, smooth scrolling, and cinematic visual effects while maintaining a clean, professional aesthetic.

## Design Philosophy
This portfolio is designed to be an immersive experience. It avoids common web development tropes by utilizing:
- **Cinematic Depth:** A dynamic, mouse-reactive lighting system with subtle noise overlays.
- **Organic Motion:** Interactive 3D floating icons representing the tech stack, responding to scroll events.
- **Fluid Typography:** High-end font pairings (Space Grotesk & Inter) for a sleek, editorial feel.
- **Frictionless Scrolling:** Implemented via Lenis for a buttery-smooth navigation experience.

---

## Tech Stack
This project is built using modern web technologies:
- **Framework:** React 19 + Vite
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion (`motion/react`)
- **Scrolling:** Lenis (Smooth Scroll)
- **Icons:** Lucide React

---

## Features
- **Interactive Hero:** Downloadable resume (CV) and direct email contact buttons.
- **Tilt Cards:** 3D hover effects on project showcases using spring physics.
- **Dynamic Background:** A custom-built reactive background with gradient meshes and spotlight tracking.
- **Scroll Reveals:** Elements gracefully fade and scale into view as the user navigates down the page.
